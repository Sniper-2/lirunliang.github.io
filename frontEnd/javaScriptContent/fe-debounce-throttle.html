    # 前端节流与防抖

    # 1.节流与防抖的定义

    • 节流：在短时间内大量触发同一个事件，那么在该事件的处理函数执行一次后，在指定时间内不再工作，过了指定时间后才重新激活，就叫做节流；

    • 防抖：在事件被触发后的n秒后再执行事件处理函数，如果再n秒内事件再一次被触发，则重新计时，就叫做防抖。

    # 2.节流的实现

      // 事件处理函数
  let runTimesInterval = 1000;   // 执行间隔  1000ms
  let lastTime = 0;
  let nowTimes = 0;
  function handleData () {
    nowTimes = new Date().getTime() 
    // 判断是否过了指定的时间间隔
    if (nowTimes - lastTime < runTimesInterval) {
      return
    }

    // 已经过了指定的时间间隔（runTimesInterval），执行处理逻辑
    lastTime = nowTimes;
