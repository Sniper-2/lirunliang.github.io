    # H5 移动端-打分页面接口

    # 1.0 名词解释

    • 动态 - 指的是其他用户所提交的给大家进行评分的动态；
    • 用户 - 指的是当前授权登录的用户

    # 2.0 获取主播动态列表的接口

    • 目前已知参数

      • 1. 动态发布者身份类型 可选值: 全部 - 用户 - 主播
      • 2. 动态发布者性别 可选值: 全部 - 男生 - 女生
      • 3. 动态类型 可选值： 照片 - 视频
      • 4. 动态 ID 可选值： 置空 - 传动态 ID 说明： 这个字段是为了用户在浏览某条动态的同时将该动态分享出去，
        通过分享出去的地址打开能够直接看到动态， 如果携带了动态 ID，请将该动态放在返回的数据列表的第一位，其他数据正常获取
      • 5.获取条数
      • 6.第几页

    • 目前已知需要接口返回的字段
